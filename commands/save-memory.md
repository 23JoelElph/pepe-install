# /save-memory

Сохрани в `~/pepe-vault/memory/` то что было в этой сессии — то что стоит унести в будущие разговоры.

**Не сохраняй:**
- Ежесекундный лог того что делал (это в git и в chat history)
- Дебаг-детали которые уже потеряли актуальность
- Промежуточные состояния, которые уже не нужны

**Сохраняй:**
- Новые факты про людей/проекты которые обнаружились
- Feedback (что не понравилось / что понравилось)
- Решения по архитектуре, которые надо помнить
- Ссылки на внешние ресурсы

Формат — как обычно (см. auto-memory instructions в system prompt): 1 факт = 1 файл с frontmatter, добавлять в MEMORY.md.

После сохранения — запусти `python3 ~/pepe-vault/memory_optimizer.py --index` чтобы обновить индекс.
