# /optimize-memory

Запусти полную оптимизацию памяти:

```bash
python3 ~/pepe-vault/memory_optimizer.py --all
```

Прочитай отчёт `/tmp/pepe-memory-report.json` и сообщи брату:
- Сколько всего узлов, сколько связей
- Дубликаты (если есть) — какие
- Orphans (файлы без входящих `[[]]`)
- Broken links — сломанные `[[wikilink]]`
- Топ-5 хабов (много исходящих)
- Топ-5 важных (много входящих)

Если есть проблемы — предложи фиксы (но не применяй без подтверждения).
